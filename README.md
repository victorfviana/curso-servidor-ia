# Curso Servidor + IA

Apresentação animada do curso **"Servidor + IA"** — Ações 1.2 e 1.3 da Frente 1 do Plano de Trabalho 2026 do **INOVAJUS Eixo III**, Justiça Federal no Ceará.

- **Acesse o curso:** abra o `index.html` (ou o link do GitHub Pages deste repositório).
- **Material de apoio:** `Banco-de-Prompts-Servidor-IA.docx` — banco de prompts para os exercícios.

## Sobre

Deck HTML autocontido (1 arquivo), com animações e modo de apresentação:

- 4 módulos (24h): Fundamentos de IA e Prompt Engineering (6h), Prática Forense (10h), Pensamento Crítico e Causal (6h), Ética, LGPD e Governança (2h).
- Navegação por slide: setas do teclado, botão de avançar, ou rolagem.
- **Pausar a animação:** botão play/pause no canto ou tecla **P** (útil ao apresentar).
- Tela cheia: tecla **F**.

Material institucional de capacitação. Uso interno da Justiça Federal no Ceará.
